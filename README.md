# Rock Paper Scissors – Python CLI Game

A simple Rock–Paper–Scissors game built using Python and played via the command line.  
The player competes against the computer using classic game rules, with a cumulative score system across a session.

## Features
- Classic Rock–Paper–Scissors gameplay
- Input validation (only accepts valid moves)
- Round-based matches (3 rounds per match)
- Cumulative session score (score continues across matches)
- Randomized computer choices
- Clear and readable CLI output

## Game Rules
- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  
- Same choices result in a draw (no score added)

## Technologies Used
- Python
- `random` library
